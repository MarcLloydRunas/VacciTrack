How to set up system application:

1. Install and open XAMPP (https://www.apachefriends.org/download.html).
2. Start MySQL and Apache.
3. Visit localhost/phpmyadmin/ on your browser.
4. Create a new database and name it "cvts_db".
5. Click on the newly made database and import the database file "cvts_db.sql" from the database folder.
6. Open "VacciTrack.exe".
